Credit Card Rewards Demo

This application was written in Angular with a SQL databasse as the "main engine" back end, which also connected to various other data sources. It sits inside the main online banking application as a module. We used Bootstrap for the layout and some of the controls, and other controls were custom made. All elements of the application,m including developing SQL scripts, C# data layer classes and a large portion of the front end. The basic functions are:

  • To select or view your quarterly rewards category. If the category has already been selected for the quarter it cannot be changed.
  • To view your spending activity and see the resulting Cash Rewards earned on each purchase. There are three categories of purchases that earn rewards, each one with a different rewards percentage, so that is displayed as well.
  • To redeem your accumulated rewards. You can have it deposited into another account, applied to your credit card balance, or donated to charity. You can also schedule redemptions to occur automatically, at a set date interval.
